============================================================
  SPAM DETECTOR — INSTALLATION and USAGE GUIDE
============================================================

REQUIREMENTS
------------
- Python 3.11 or 3.12 
- VS Code (or preferred IDE)              
- Ollama (running locally)
- Llama 3.1 model (pulled via Ollama)

------------------------------------------------------------
STEP 1 — CLONE / DOWNLOAD THE PROJECT
------------------------------------------------------------
Download or unzip the project folder so you have:

  spam-detector/
  ├── data/
  │   ├── ham500.txt
  │   ├── nigerian_format80.txt
  │   ├── nigerian_format400.txt
  │   └── spam100.txt
  ├── src/
  │   ├── parse_dataset.py
  │   ├── classifier.py
  │   └── task5_scam_reply_generator.py
  ├── requirements.txt
  └── README.txt

------------------------------------------------------------
STEP 2 — OPEN THE PROJECT IN VS CODE
------------------------------------------------------------
  Open Folder and select the spam-detector folder

------------------------------------------------------------
STEP 3 — CREATE A VIRTUAL ENVIRONMENT
------------------------------------------------------------
Open the terminal in VS Code (Ctrl + `)  and run:

  python -m venv venv

This creates an isolated Python environment for the project.

------------------------------------------------------------
STEP 4 — ACTIVATE THE VIRTUAL ENVIRONMENT
------------------------------------------------------------
Windows:
  venv\Scripts\activate

Mac/Linux:
  source venv/bin/activate

You should see (venv) appear at the start of your prompt.

NOTE: You need to activate the venv every time you open
a new terminal session.

------------------------------------------------------------
STEP 5 — INSTALL DEPENDENCIES
------------------------------------------------------------
With the venv active, run:

  pip install -r requirements.txt

This installs all required packages (scikit-learn, pandas,
matplotlib, seaborn, spacy, etc.).

Next, download the required spaCy language model by running:

  python -m spacy download en_core_web_sm

------------------------------------------------------------
STEP 6 — SELECT THE PYTHON INTERPRETER IN VS CODE
------------------------------------------------------------
Press Ctrl+Shift+P and type "Python: Select Interpreter"
 choose the one that says:

  .\venv\Scripts\python.exe

So VS Code uses the projects venv

------------------------------------------------------------
STEP 7 — PARSE THE DATASET
------------------------------------------------------------
Run from the spam-detector/ folder:

  python src/parse_dataset.py

Expected output:
  Parsing dataset...
    Loaded  500 emails  (label=0)  <- ham500.txt
    Loaded  100 emails  (label=1)  <- spam100.txt
    Loaded  400 emails  (label=1)  <- nigerian_format400.txt
    Loaded   80 emails  (label=1)  <- nigerian_format80.txt
  Total emails : 1080
  Saved to data\emails.csv

------------------------------------------------------------
STEP 8 — TRAIN AND EVALUATE THE CLASSIFIER
------------------------------------------------------------
Run:
  python src/classifier.py

This will:
  Train two models: Naive Bayes and Linear SVM
  Run 5-fold cross-validation
  Print accuracy, precision, recall, F1 for each model for evaluation
  Save confusion matrix plots to models/
  Save the best model to models/best_model.joblib

------------------------------------------------------------
STEP 9 — START OLLAMA & PULL THE MODEL
------------------------------------------------------------
Before generating AI replies, you must ensure Ollama is running 
on your machine. 

Open a new, separate terminal (or command prompt) and run:

  ollama run llama3.1

This will start the local server and pull the Llama 3.1 model 
if you haven't downloaded it yet. Leave this running in the 
background.

------------------------------------------------------------
STEP 10 — GENERATE SCAM-BAITING REPLIES
------------------------------------------------------------
Run from the spam-detector/ folder (in your original terminal):

  python src/task5_scam_reply_generator.py

This will load the best model and allow you to select:
  1 - Auto mode (random emails from dataset)
  2 - Interactive mode (paste your own email)

*NOTE: Both modes support Multi-Turn conversations. You can 
act as the attacker and chat back and forth with the AI personas 
(Theodore or Patricia) to see how they maintain character.*
